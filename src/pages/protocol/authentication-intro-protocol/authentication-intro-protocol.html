认证说明
认证有几种类型？

认证主要分为个人和组织两种，目前已经开放了个人身份的认证。认证时，您需要选中一个主要身份，完善资料后，您还可以选择多个次要身份。

我需要多久能知道认证结果？

我们会在3个工作日内完成审核，并以站内信的方式告知您。

认证后有什么作用？

1. 您的认证图标将被点亮

2. 您将具有发起直播等更多的功能权限

3. 系统能为您更有效地推荐有价值的内容

4. 为您提供更多被推荐的机会

认证后的信息可以修改吗？

您认证后的信息不能进行修改，但您可以通过再次申请认证的方式来更新您的信息

个人隐私会被别人看到吗？

您在认证过程中填写的如身份证号、照片等隐私信息，都会保密，不会被其他用户看到
